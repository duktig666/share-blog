# Flinkз®Җд»Ӣ

Apache Flink жҳҜдёҖдёӘжЎҶжһ¶е’ҢеҲҶеёғејҸеӨ„зҗҶеј•ж“ҺпјҢз”ЁдәҺеҜ№ ж— з•Ңе’Ңжңүз•Ңж•°жҚ®жөҒ иҝӣиЎҢзҠ¶жҖҒи®Ўз®—гҖӮ

# дёәд»Җд№ҲиҰҒдҪҝз”ЁFlinkпјҹ

  • жөҒж•°жҚ®жӣҙзңҹе®һең°еҸҚжҳ дәҶжҲ‘д»¬зҡ„з”ҹжҙ»ж–№ејҸ
  • дј з»ҹзҡ„ж•°жҚ®жһ¶жһ„жҳҜеҹәдәҺжңүйҷҗж•°жҚ®йӣҶзҡ„
  • зӣ®ж Ү
    • дҪҺе»¶иҝҹпјҲSpark Streaming зҡ„е»¶иҝҹжҳҜз§’зә§пјҢFlink е»¶иҝҹжҳҜжҜ«з§’зә§пјү
    • й«ҳеҗһеҗҗпјҲйҳҝйҮҢжҜҸз§’й’ҹдҪҝз”Ё Flink еӨ„зҗҶ 4.6PBпјҢеҸҢеҚҒдёҖеӨ§еұҸпјү
    • з»“жһңзҡ„еҮҶзЎ®жҖ§е’ҢиүҜеҘҪзҡ„е®№й”ҷжҖ§пјҲexactly-onceпјү

# жөҒж•°жҚ®дҪҝз”ЁеңәжҷҜ

з”өе•Ҷе’ҢеёӮеңәиҗҘй”Җпјҡж•°жҚ®жҠҘиЎЁгҖҒе№ҝе‘ҠжҠ•ж”ҫгҖҒдёҡеҠЎжөҒзЁӢйңҖиҰҒ

зү©иҒ”зҪ‘пјҲIOTпјүпјҡдј ж„ҹеҷЁе®һж—¶ж•°жҚ®йҮҮйӣҶе’ҢжҳҫзӨәгҖҒе®һж—¶жҠҘиӯҰпјҢдәӨйҖҡиҝҗиҫ“дёҡпјҲиҮӘеҠЁ й©ҫй©¶пјү

з”өдҝЎдёҡпјҡеҹәз«ҷжөҒйҮҸи°ғй…Қ

й“¶иЎҢе’ҢйҮ‘иһҚдёҡпјҡе®һж—¶з»“з®—е’ҢйҖҡзҹҘжҺЁйҖҒпјҢе®һж—¶жЈҖжөӢејӮеёёиЎҢдёәпјҲдҝЎз”ЁеҚЎзӣ—еҚЎпјү

# Flinkзҡ„иҝҗиЎҢжһ¶жһ„

# дј з»ҹж•°жҚ®еӨ„зҗҶжһ¶жһ„

дәӢеҠЎеӨ„зҗҶпјҲOLTPпјүпјҡ

еҲҶжһҗеӨ„зҗҶпјҲOLAPпјүпјҡ

е°Ҷж•°жҚ®д»ҺдёҡеҠЎж•°жҚ®еә“еӨҚеҲ¶еҲ°ж•°д»“пјҢеҶҚиҝӣиЎҢеҲҶжһҗе’ҢжҹҘиҜў

# жңүзҠ¶жҖҒзҡ„жөҒејҸеӨ„зҗҶ

lambda жһ¶жһ„пјҡз”ЁдёӨеҘ—зі»з»ҹпјҢеҗҢж—¶дҝқиҜҒдҪҺе»¶иҝҹе’Ңз»“жһңеҮҶзЎ®

# жөҒеӨ„зҗҶзҡ„жј”еҸҳ

# Flinkзҡ„дё»иҰҒзү№зӮ№

дәӢд»¶й©ұеҠЁ

еҹәдәҺжөҒзҡ„дё–з•Ңи§Ӯ

еңЁ Flink зҡ„дё–з•Ңи§ӮдёӯпјҢдёҖеҲҮйғҪжҳҜз”ұжөҒз»„жҲҗзҡ„пјҢзҰ»зәҝж•°жҚ®жҳҜжңүз•Ңзҡ„жөҒпјӣе®һж—¶ж•°жҚ®жҳҜдёҖдёӘжІЎжңүз•Ңйҷҗзҡ„жөҒпјҡиҝҷе°ұжҳҜжүҖи°“зҡ„жңүз•ҢжөҒе’Ңж— з•ҢжөҒ

Flink зҡ„еҲҶеұӮ API

и¶ҠйЎ¶еұӮи¶ҠжҠҪиұЎпјҢиЎЁиҫҫеҗ«д№үи¶Ҡз®ҖжҳҺпјҢдҪҝз”Ёи¶Ҡж–№дҫҝ

и¶Ҡеә•еұӮи¶Ҡе…·дҪ“пјҢиЎЁиҫҫиғҪеҠӣи¶Ҡдё°еҜҢпјҢдҪҝз”Ёи¶ҠзҒөжҙ»

е…¶д»–пјҡ

  • ж”ҜжҢҒдәӢд»¶ж—¶й—ҙпјҲevent-timeпјүе’ҢеӨ„зҗҶж—¶й—ҙпјҲprocessing-timeпјүиҜӯд№ү
  • зІҫзЎ®дёҖж¬ЎпјҲexactly-onceпјүзҡ„зҠ¶жҖҒдёҖиҮҙжҖ§дҝқиҜҒ
  • дҪҺе»¶иҝҹпјҢжҜҸз§’еӨ„зҗҶж•°зҷҫдёҮдёӘдәӢд»¶пјҢжҜ«з§’зә§е»¶иҝҹпјҲе®һйҷ…дёҠе°ұжҳҜжІЎжңүе»¶иҝҹпјү
  • дёҺдј—еӨҡеёёз”ЁеӯҳеӮЁзі»з»ҹзҡ„иҝһжҺҘпјҲESпјҢHBaseпјҢMySQLпјҢRedisвӢҜпјү
  • й«ҳеҸҜз”ЁпјҲzookeeperпјүпјҢеҠЁжҖҒжү©еұ•пјҢе®һзҺ° 7*24 е°Ҹж—¶е…ЁеӨ©еҖҷиҝҗиЎҢ

# Flinkе’ҢSpark Streamingзҡ„еҢәеҲ«

  • жөҒпјҲstreamпјүе’Ңеҫ®жү№
  • ж•°жҚ®жЁЎеһӢ
    • Spark йҮҮз”Ё RDD жЁЎеһӢпјҢSpark Streaming зҡ„ DStream е®һйҷ…дёҠд№ҹе°ұжҳҜдёҖз»„з»„е°Ҹжү№ж•°жҚ® RDD зҡ„йӣҶеҗҲ
    • Flink еҹәжң¬ж•°жҚ®жЁЎеһӢжҳҜж•°жҚ®жөҒпјҢд»ҘеҸҠдәӢд»¶пјҲEventпјүеәҸеҲ—пјҲIntegerгҖҒStringгҖҒLongгҖҒPOJO Classпјү
  • иҝҗиЎҢж—¶жһ¶жһ„
    • Spark жҳҜжү№и®Ўз®—пјҢе°Ҷ DAG еҲ’еҲҶдёәдёҚеҗҢзҡ„ StageпјҢдёҖдёӘ Stageе®ҢжҲҗеҗҺжүҚеҸҜд»Ҙи®Ўз®—дёӢдёҖдёӘ Stage
    • Flink жҳҜж ҮеҮҶзҡ„жөҒжү§иЎҢжЁЎејҸпјҢдёҖдёӘдәӢд»¶еңЁдёҖдёӘиҠӮзӮ№еӨ„зҗҶе®ҢеҗҺеҸҜд»ҘзӣҙжҺҘеҸ‘еҫҖдёӢдёҖдёӘиҠӮзӮ№иҝӣиЎҢеӨ„зҗҶ

# Flink иҝҗиЎҢж—¶зҡ„з»„д»¶

# дҪңдёҡз®ЎзҗҶеҷЁ (JobManager)

жҺ§еҲ¶дёҖдёӘеә”з”ЁзЁӢеәҸжү§иЎҢзҡ„дё»иҝӣзЁӢпјҢд№ҹе°ұжҳҜиҜҙпјҢжҜҸдёӘеә”з”ЁзЁӢеәҸйғҪдјҡиў«дёҖдёӘдёҚеҗҢзҡ„ JobManager жүҖжҺ§еҲ¶жү§иЎҢгҖӮ

JobManager дјҡе…ҲжҺҘж”¶еҲ°иҰҒжү§иЎҢзҡ„еә”з”ЁзЁӢеәҸпјҢиҝҷдёӘеә”з”ЁзЁӢеәҸдјҡеҢ…жӢ¬пјҡдҪңдёҡеӣҫ (JobGraph)гҖҒйҖ»иҫ‘ж•°жҚ®жөҒеӣҫе’Ңжү“еҢ…дәҶжүҖжңүзҡ„зұ»гҖҒеә“е’Ңе…¶е®ғиө„жәҗзҡ„ JAR еҢ…гҖӮ

JobManager дјҡжҠҠ JobGraph иҪ¬жҚўжҲҗдёҖдёӘзү©зҗҶеұӮйқўзҡ„ж•°жҚ®жөҒеӣҫпјҢиҝҷдёӘеӣҫиў«еҸ«еҒҡвҖңжү§иЎҢеӣҫвҖқ(ExecutionGraph)пјҢеҢ…еҗ«дәҶжүҖжңүеҸҜд»Ҙе№¶еҸ‘жү§иЎҢзҡ„д»»еҠЎгҖӮ

JobManager дјҡеҗ‘иө„жәҗз®ЎзҗҶеҷЁ (Flink зҡ„иө„жәҗз®ЎзҗҶеҷЁ) иҜ·жұӮжү§иЎҢд»»еҠЎеҝ…иҰҒзҡ„иө„жәҗпјҢд№ҹе°ұжҳҜд»»еҠЎз®ЎзҗҶеҷЁ (TaskManager) дёҠзҡ„д»»еҠЎжҸ’ж§ҪпјҲslotпјүгҖӮдёҖж—Ұе®ғиҺ·еҸ–еҲ°дәҶи¶іеӨҹзҡ„иө„жәҗпјҢе°ұдјҡе°Ҷжү§иЎҢеӣҫ (DAG) еҲҶеҸ‘еҲ°зңҹжӯЈиҝҗиЎҢе®ғд»¬зҡ„ TaskManager дёҠгҖӮиҖҢеңЁиҝҗиЎҢиҝҮзЁӢдёӯпјҢJobManager дјҡиҙҹиҙЈжүҖжңүйңҖиҰҒдёӯеӨ®еҚҸи°ғзҡ„ж“ҚдҪңпјҢжҜ”еҰӮиҜҙжЈҖжҹҘзӮ№ (checkpoints) зҡ„еҚҸи°ғгҖӮ

# иө„жәҗз®ЎзҗҶеҷЁ (ResourceManager)

дё»иҰҒиҙҹиҙЈз®ЎзҗҶд»»еҠЎз®ЎзҗҶеҷЁ(TaskManager)зҡ„жҸ’ж§Ҫ(slot) ,TaskManagerжҸ’ж§ҪжҳҜFlinkдёӯе®ҡд№үзҡ„еӨ„зҗҶиө„жәҗеҚ•е…ғгҖӮ

FlinkдёәдёҚеҗҢзҡ„зҺҜеўғе’Ңиө„жәҗз®ЎзҗҶе·Ҙе…·жҸҗдҫӣдәҶдёҚеҗҢиө„жәҗз®ЎзҗҶеҷЁпјҢжҜ”еҰӮYARNгҖҒMesosгҖҒKubernetes(з®ЎзҗҶdockerе®№еҷЁз»„жҲҗзҡ„йӣҶзҫӨ)пјҢд»ҘеҸҠStandalone(зӢ¬з«ӢйӣҶзҫӨ)йғЁзҪІгҖӮ

еҪ“JobManager з”іиҜ·жҸ’ж§Ҫиө„жәҗж—¶пјҢFlinkзҡ„иө„жәҗз®ЎзҗҶеҷЁдјҡе°Ҷжңүз©әй—ІжҸ’ж§Ҫзҡ„TaskManager еҲҶй…Қз»ҷJobManagerгҖӮеҰӮжһңFlinkзҡ„иө„жәҗз®ЎзҗҶеҷЁжІЎжңүи¶іеӨҹзҡ„жҸ’ж§ҪжқҘж»Ўи¶іJobManager зҡ„иҜ·жұӮпјҢе®ғиҝҳеҸҜд»Ҙеҗ‘Yarnзҡ„иө„жәҗз®ЎзҗҶеҷЁеҸ‘иө·дјҡиҜқпјҢд»ҘжҸҗдҫӣеҗҜеҠЁTaskManagerиҝӣзЁӢзҡ„е®№еҷЁгҖӮ

# еҲҶеҸ‘еҷЁ (Dispatcher)

еҸҜд»Ҙи·ЁдҪңдёҡиҝҗиЎҢпјҢе®ғдёәеә”з”ЁжҸҗдәӨжҸҗдҫӣдәҶRESTfulжҺҘеҸЈ(GET/PUT/DELETE/POST)гҖӮ

еҪ“дёҖдёӘеә”з”Ёиў«жҸҗдәӨжү§иЎҢж—¶пјҢеҲҶеҸ‘еҷЁе°ұдјҡеҗҜеҠЁе№¶е°Ҷеә”з”Ёз§»дәӨз»ҷдёҖдёӘJobManagerгҖӮ

Dispatcher д№ҹдјҡеҗҜеҠЁдёҖдёӘ Web Ul(localhost:8081)пјҢз”ЁжқҘж–№дҫҝең°еұ•зӨәе’Ңзӣ‘жҺ§дҪңдёҡиЎҢзҡ„дҝЎжҒҜгҖӮ

Dispatcher еңЁжһ¶жһ„дёӯеҸҜиғҪе№¶дёҚжҳҜеҝ…йңҖзҡ„пјҢиҝҷеҸ–еҶідәҺеә”з”ЁжҸҗдәӨиҝҗиЎҢзҡ„ж–№ејҸгҖӮ

# JobMaster

JobMaster иҙҹиҙЈз®ЎзҗҶеҚ•дёӘ JobGraph зҡ„жү§иЎҢгҖӮFlink йӣҶзҫӨдёӯеҸҜд»ҘеҗҢж—¶иҝҗиЎҢеӨҡдёӘдҪңдёҡпјҢжҜҸдёӘдҪңдёҡйғҪжңүиҮӘе·ұзҡ„JobMasterгҖӮ

# TaskManager е’Ң Slots

Flink дёӯжҜҸдёҖдёӘ TaskManager йғҪжҳҜдёҖдёӘ JVM иҝӣзЁӢпјҢжҜҸдёҖдёӘд»»еҠЎжҸ’ж§ҪйғҪдјҡеҗҜеҠЁдёҖдёӘзәҝзЁӢпјҢе®ғеҸҜиғҪдјҡеңЁзӢ¬з«Ӣзҡ„зәҝзЁӢдёҠжү§иЎҢдёҖдёӘжҲ–еӨҡдёӘ subtaskпјҢжҜҸдёҖдёӘеӯҗд»»еҠЎеҚ з”ЁдёҖдёӘд»»еҠЎжҸ’ж§ҪпјҲTask Slotпјү

дёәдәҶжҺ§еҲ¶дёҖдёӘ TaskManager иғҪжҺҘж”¶еӨҡе°‘дёӘ taskпјҢTaskManager йҖҡиҝҮ task slot жқҘиҝӣиЎҢжҺ§еҲ¶пјҲдёҖдёӘ TaskManagerиҮіе°‘жңүдёҖдёӘ slotпјү

й»ҳи®Өжғ…еҶөдёӢпјҢFlink е…Ғи®ёеӯҗд»»еҠЎе…ұдә« slotгҖӮиҝҷж ·зҡ„з»“жһңжҳҜпјҢдёҖдёӘ slot еҸҜд»ҘдҝқеӯҳдҪңдёҡзҡ„ж•ҙдёӘз®ЎйҒ“гҖӮ

Task Slot жҳҜйқҷжҖҒзҡ„жҰӮеҝөпјҢжҳҜжҢҮ TaskManager е…·жңүзҡ„е№¶еҸ‘жү§иЎҢиғҪеҠӣгҖӮ

# зЁӢеәҸдёҺж•°жҚ®жөҒ (DataFlow)

жүҖжңүзҡ„ Flink зЁӢеәҸйғҪжҳҜз”ұдёүйғЁеҲҶз»„жҲҗзҡ„пјҡSourceгҖҒTransformation е’Ң SinkгҖӮ

Source иҙҹиҙЈиҜ»еҸ–ж•°жҚ®жәҗпјҢTransformation еҲ©з”Ёеҗ„з§Қз®—еӯҗиҝӣиЎҢеӨ„зҗҶеҠ е·ҘпјҢSink иҙҹиҙЈиҫ“еҮәгҖӮ

еңЁиҝҗиЎҢж—¶пјҢFlink дёҠиҝҗиЎҢзҡ„зЁӢеәҸдјҡиў«жҳ е°„жҲҗвҖңйҖ»иҫ‘ж•°жҚ®жөҒвҖқпјҲdataflowsпјүпјҢе®ғеҢ…еҗ«дәҶиҝҷдёүйғЁеҲҶ

жҜҸдёҖдёӘ dataflow д»ҘдёҖдёӘжҲ–еӨҡдёӘ sources ејҖе§Ӣд»ҘдёҖдёӘжҲ–еӨҡдёӘsinks з»“жқҹгҖӮdataflow зұ»дјјдәҺд»»ж„Ҹзҡ„жңүеҗ‘ж— зҺҜеӣҫпјҲDAGпјү

еңЁеӨ§йғЁеҲҶжғ…еҶөдёӢпјҢзЁӢеәҸдёӯзҡ„иҪ¬жҚўиҝҗз®—пјҲtransformationsпјүи·ҹdataflow дёӯзҡ„з®—еӯҗпјҲoperatorпјүжҳҜдёҖдёҖеҜ№еә”зҡ„е…ізі»

# еӣҫж•°жҚ®з»“жһ„зҡ„иҪ¬еҢ–

  • StreamGraphпјҡжҳҜж №жҚ®з”ЁжҲ·йҖҡиҝҮ Stream API зј–еҶҷзҡ„д»Јз Ғз”ҹжҲҗзҡ„жңҖеҲқзҡ„еӣҫгҖӮз”ЁжқҘиЎЁзӨәзЁӢеәҸзҡ„жӢ“жү‘з»“жһ„гҖӮ
  • JobGraphпјҡStreamGraph еңЁзј–иҜ‘зҡ„йҳ¶ж®өз»ҸиҝҮдјҳеҢ–еҗҺз”ҹжҲҗдәҶJobGraphпјҢжҸҗдәӨз»ҷ JobManager зҡ„ж•°жҚ®з»“жһ„гҖӮдё»иҰҒзҡ„дјҳеҢ–дёәпјҢе°ҶеӨҡдёӘз¬ҰеҗҲжқЎд»¶пјҲзӘ„дҫқиө–пјҢжІЎжңү shuffleпјүзҡ„з®—еӯҗ chain еңЁдёҖиө·дҪңдёәдёҖдёӘиҠӮзӮ№гҖӮ
  • ExecutionGraphпјҡJobManager ж №жҚ® JobGraph з”ҹжҲҗExecutionGraphгҖӮExecutionGraph жҳҜ JobGraph зҡ„е№¶иЎҢеҢ–зүҲжң¬пјҢжҳҜи°ғеәҰеұӮжңҖж ёеҝғзҡ„ж•°жҚ®з»“жһ„гҖӮ
  • зү©зҗҶжү§иЎҢеӣҫпјҡJobManager ж №жҚ® ExecutionGraph еҜ№ Job иҝӣиЎҢи°ғеәҰеҗҺпјҢеңЁеҗ„дёӘ TaskManager дёҠйғЁзҪІ Task еҗҺеҪўжҲҗзҡ„вҖңеӣҫвҖқпјҢе№¶дёҚжҳҜдёҖдёӘе…·дҪ“зҡ„ж•°жҚ®з»“жһ„гҖӮ

# е№¶иЎҢеәҰ

  • дёҖдёӘзү№е®ҡз®—еӯҗзҡ„еӯҗд»»еҠЎпјҲsubtaskпјүзҡ„дёӘж•°иў«з§°д№Ӣдёәе…¶е№¶иЎҢеәҰпјҲparallelismпјүгҖӮдёҖиҲ¬жғ…еҶөдёӢпјҢдёҖдёӘ stream зҡ„е№¶иЎҢеәҰпјҢеҸҜд»Ҙи®Өдёәе°ұжҳҜе…¶жүҖжңүз®—еӯҗдёӯжңҖеӨ§зҡ„е№¶иЎҢеәҰгҖӮ
  • з®—еӯҗд№Ӣй—ҙдј иҫ“ж•°жҚ®зҡ„еҪўејҸеҸҜд»ҘжҳҜ one-to-one (forwarding) зҡ„жЁЎејҸд№ҹеҸҜд»ҘжҳҜ redistributing зҡ„жЁЎејҸпјҢе…·дҪ“жҳҜе“ӘдёҖз§ҚеҪўејҸпјҢеҸ–еҶідәҺз®—еӯҗзҡ„з§Қзұ»
    • One-to-oneпјҡstream з»ҙжҠӨзқҖеҲҶеҢәд»ҘеҸҠе…ғзҙ зҡ„йЎәеәҸпјҲжҜ”еҰӮsource е’Ң map д№Ӣй—ҙпјүгҖӮиҝҷж„Ҹе‘ізқҖ map з®—еӯҗзҡ„еӯҗд»»еҠЎзңӢеҲ°зҡ„е…ғзҙ зҡ„дёӘж•°д»ҘеҸҠйЎәеәҸи·ҹ source з®—еӯҗзҡ„еӯҗд»»еҠЎз”ҹдә§зҡ„е…ғзҙ зҡ„дёӘж•°гҖҒйЎәеәҸзӣёеҗҢгҖӮmapгҖҒfilterгҖҒflatMap зӯүз®—еӯҗйғҪжҳҜ one-to-oneзҡ„еҜ№еә”е…ізі»гҖӮ
    • Redistributingпјҡstream зҡ„еҲҶеҢәдјҡеҸ‘з”ҹж”№еҸҳгҖӮжҜҸдёҖдёӘз®—еӯҗзҡ„еӯҗд»»еҠЎдҫқжҚ®жүҖйҖүжӢ©зҡ„ transformation еҸ‘йҖҒж•°жҚ®еҲ°дёҚеҗҢзҡ„зӣ®ж Үд»»еҠЎгҖӮдҫӢеҰӮпјҢkeyBy еҹәдәҺ hashCode йҮҚеҲҶеҢәгҖҒиҖҢ broadcast е’Ңrebalance дјҡйҡҸжңәйҮҚж–°еҲҶеҢәпјҢиҝҷдәӣз®—еӯҗйғҪдјҡеј•иө· redistribute иҝҮзЁӢпјҢиҖҢ redistribute иҝҮзЁӢе°ұзұ»дјјдәҺ Spark дёӯзҡ„ shuffle иҝҮзЁӢгҖӮ

public class CustomSourceExample {

    public static void main(String[] args) throws Exception {
        StreamExecutionEnvironment env = StreamExecutionEnvironment.getExecutionEnvironment();
        env.setParallelism(1);
		
        // и®ҫзҪ®иҮӘе®ҡд№үзҡ„ж•°жҚ®жәҗ
        DataStreamSource<Event> stream = env.addSource(new ClickSource());

        stream.print();

        env.execute();
    }

    /**
     * SourceFunctionе№¶иЎҢеәҰеҸӘиғҪдёә1
     * иҮӘе®ҡд№үе№¶иЎҢеҢ–зүҲжң¬зҡ„ж•°жҚ®жәҗпјҢйңҖиҰҒдҪҝз”ЁParallelSourceFunction
     */
    public static class ClickSource implements SourceFunction<Event> {
        private boolean running = true;
        private String[] userArr = {"Mary", "Bob", "Alice", "Liz"};
        private String[] urlArr = {"./home", "./cart", "./fav", "./prod?id=1", "./prod?id=2"};
        private Random random = new Random();

        @Override
        public void run(SourceContext<Event> ctx) throws Exception {
            while (running) {
                // collectж–№жі•пјҢеҗ‘дёӢжёёеҸ‘йҖҒж•°жҚ®
                ctx.collect(
                        new Event(
                                userArr[random.nextInt(userArr.length)],
                                urlArr[random.nextInt(urlArr.length)],
                                Calendar.getInstance().getTimeInMillis()
                        )
                );
                Thread.sleep(1000L);
            }
        }

        @Override
        public void cancel() {
            running = false;
        }
    }

    public static class Event {
        public String user;
        public String url;
        public Long timestamp;

        public Event() {
        }

        public Event(String user, String url, Long timestamp) {
            this.user = user;
            this.url = url;
            this.timestamp = timestamp;
        }

        @Override
        public String toString() {
            return "Event{" +
                    "user='" + user + '\'' +
                    ", url='" + url + '\'' +
                    ", timestamp=" + new Timestamp(timestamp) +
                    '}';
        }
    }

}

            out.collect(value);
        }
    }
}

KeyedStream еҸҜд»ҘдҪҝз”ЁmapпјҢп¬ӮatMap е’Ңп¬Ғlter з®—еӯҗжқҘеӨ„зҗҶгҖӮдҪҝз”ЁkeyBy з®—еӯҗжқҘе°ҶDataStream иҪ¬жҚўжҲҗKeyedStreamпјҢеҹәдәҺkey зҡ„иҪ¬жҚўж“ҚдҪңпјҡж»ҡеҠЁиҒҡеҗҲе’Ңreduceз®—еӯҗгҖӮ

# KEYBY

keyBy йҖҡиҝҮжҢҮе®ҡ key жқҘе°Ҷ DataStream иҪ¬жҚўжҲҗ KeyedStreamгҖӮеҹәдәҺдёҚеҗҢзҡ„ keyпјҢжөҒдёӯзҡ„дәӢд»¶е°Ҷиў«еҲҶй…ҚеҲ°дёҚеҗҢзҡ„еҲҶеҢәдёӯеҺ»гҖӮжүҖжңүе…·жңүзӣёеҗҢ key зҡ„дәӢд»¶е°ҶдјҡеңЁжҺҘдёӢжқҘзҡ„ж“ҚдҪңз¬Ұзҡ„еҗҢдёҖдёӘеӯҗд»»еҠЎж§ҪдёӯиҝӣиЎҢеӨ„зҗҶгҖӮжӢҘжңүдёҚеҗҢ key зҡ„дәӢд»¶еҸҜд»ҘеңЁеҗҢдёҖдёӘд»»еҠЎдёӯеӨ„зҗҶгҖӮдҪҶжҳҜз®—еӯҗеҸӘиғҪи®ҝй—®еҪ“еүҚдәӢд»¶зҡ„key жүҖеҜ№еә”зҡ„зҠ¶жҖҒгҖӮ

KeyedStream<UserBehavior, String> keyed = stream.keyBy(r -> r.itemId);

stream.keyBy(new KeySelector<UserBehavior, String>() {
        @Override
        public String getKey(UserBehavior value) throws Exception {
            return value.itemId;
        }
    });

# ж»ҡеҠЁиҒҡеҗҲ

ж»ҡеҠЁиҒҡеҗҲз®—еӯҗз”ұKeyedStream и°ғз”ЁпјҢе№¶з”ҹжҲҗдёҖдёӘиҒҡеҗҲд»ҘеҗҺзҡ„DataStreamпјҢдҫӢеҰӮпјҡsumпјҢminimumпјҢmaximumгҖӮдёҖдёӘж»ҡеҠЁиҒҡеҗҲз®—еӯҗдјҡдёәжҜҸдёҖдёӘи§ӮеҜҹеҲ°зҡ„key дҝқеӯҳдёҖдёӘиҒҡеҗҲзҡ„еҖјгҖӮй’ҲеҜ№жҜҸдёҖдёӘиҫ“е…ҘдәӢд»¶пјҢз®—еӯҗе°Ҷдјҡжӣҙж–°дҝқеӯҳзҡ„иҒҡеҗҲз»“жһңпјҢе№¶еҸ‘йҖҒдёҖдёӘеёҰжңүжӣҙж–°еҗҺзҡ„еҖјзҡ„дәӢд»¶еҲ°дёӢжёёз®—еӯҗгҖӮж»ҡеҠЁиҒҡеҗҲдёҚйңҖиҰҒз”ЁжҲ·иҮӘе®ҡд№үеҮҪж•°пјҢдҪҶйңҖиҰҒжҺҘеҸ—дёҖдёӘеҸӮж•°пјҢиҝҷдёӘеҸӮж•°жҢҮе®ҡдәҶ еңЁе“ӘдёҖдёӘеӯ—ж®өдёҠйқўеҒҡиҒҡеҗҲж“ҚдҪңгҖӮDataStream API жҸҗдҫӣдәҶд»ҘдёӢж»ҡеҠЁиҒҡеҗҲж–№жі•гҖӮ

  • sum()пјҡеңЁиҫ“е…ҘжөҒдёҠеҜ№жҢҮе®ҡзҡ„еӯ—ж®өеҒҡж»ҡеҠЁзӣёеҠ ж“ҚдҪңгҖӮ
  • min()пјҡеңЁиҫ“е…ҘжөҒдёҠеҜ№жҢҮе®ҡзҡ„еӯ—ж®өжұӮжңҖе°ҸеҖјгҖӮ
  • max()пјҡеңЁиҫ“е…ҘжөҒдёҠеҜ№жҢҮе®ҡзҡ„еӯ—ж®өжұӮжңҖеӨ§еҖјгҖӮ
  • minBy()пјҡеңЁиҫ“е…ҘжөҒдёҠй’ҲеҜ№жҢҮе®ҡеӯ—ж®өжұӮжңҖе°ҸеҖјпјҢе№¶иҝ”еӣһеҢ…еҗ«еҪ“еүҚи§ӮеҜҹеҲ°зҡ„жңҖе°ҸеҖјзҡ„дәӢд»¶гҖӮ
  • maxBy()пјҡеңЁиҫ“е…ҘжөҒдёҠй’ҲеҜ№жҢҮе®ҡеӯ—ж®өжұӮжңҖеӨ§еҖјпјҢе№¶иҝ”еӣһеҢ…еҗ«еҪ“еүҚи§ӮеҜҹеҲ°зҡ„жңҖеӨ§еҖјзҡ„дәӢд»¶гҖӮ

ж»ҡеҠЁиҒҡеҗҲз®—еӯҗж— жі•з»„еҗҲиө·жқҘдҪҝз”ЁпјҢжҜҸж¬Ўи®Ўз®—еҸӘиғҪдҪҝз”ЁдёҖдёӘеҚ•зӢ¬зҡ„ж»ҡеҠЁиҒҡеҗҲз®—еӯҗгҖӮ

дёӢйқўзҡ„дҫӢеӯҗж №жҚ®з¬¬дёҖдёӘеӯ—ж®өжқҘеҜ№зұ»еһӢдёә Tuple3<Int, Int, Int> зҡ„жөҒеҒҡеҲҶжөҒж“ҚдҪңпјҢз„¶еҗҺ й’ҲеҜ№з¬¬дәҢдёӘеӯ—ж®өеҒҡж»ҡеҠЁжұӮе’Ңж“ҚдҪңгҖӮ

DataStream<Tuple3<Integer, Integer, Integer>> resultStream = inputStream

# REDUCE

reduce з®—еӯҗжҳҜж»ҡеҠЁиҒҡеҗҲзҡ„жіӣеҢ–е®һзҺ°гҖӮе®ғе°ҶдёҖдёӘ ReduceFunction еә”з”ЁеҲ°дәҶдёҖдёӘ KeyedStream дёҠйқўеҺ»гҖӮreduce з®—еӯҗе°ҶдјҡжҠҠжҜҸдёҖдёӘиҫ“е…ҘдәӢд»¶е’ҢеҪ“еүҚе·Із»Ҹ reduce еҮәжқҘзҡ„еҖјеҒҡиҒҡеҗҲи®Ўз®—гҖӮreduce ж“ҚдҪңдёҚдјҡж”№еҸҳжөҒзҡ„дәӢд»¶зұ»еһӢгҖӮиҫ“еҮәжөҒж•°жҚ®зұ»еһӢе’Ңиҫ“е…ҘжөҒж•°жҚ®зұ»еһӢжҳҜдёҖж ·зҡ„гҖӮ

reduce еҮҪж•°еҸҜд»ҘйҖҡиҝҮе®һзҺ°жҺҘеҸЈ ReduceFunction жқҘеҲӣе»әдёҖдёӘзұ»гҖӮReduceFunction жҺҘеҸЈе®ҡд№үдәҶ reduce() ж–№жі•пјҢжӯӨж–№жі•жҺҘж”¶дёӨдёӘиҫ“е…ҘдәӢд»¶пјҢиҫ“еҮәдёҖдёӘзӣёеҗҢзұ»еһӢзҡ„дәӢд»¶гҖӮ

inputStream
    .keyBy(r -> r.f0)
    .reduce(new ReduceFunction<Tuple3<Integer, Integer, Integer>>() {
        @Override
        public Tuple3<Integer, Integer, Integer> reduce(Tuple3<Integer, Integer,Integer> value1, Tuple3<Integer, Integer, Integer> value2) throws Exception{
            if (value1.f1 > value2.f1) {
                return value1;
            } else {
                return value2;
            }
        }
    }).print();
