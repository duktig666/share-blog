44、查询和“李军”同性别并同班的同学Sname.

SELECT sname
FROM student
WHERE (ssex, class) = (SELECT ssex, class FROM student WHREE sname = '李军');

45、查询所有选修“计算机导论”课程的“男”同学的成绩表

SELECT *
FROM score,
     student
WHERE score.SNO = student.SNO
  and ssex = '男'
  and cno = (
    SELECT cno
    FROM course
    WHERE cname = '计算机导论');
